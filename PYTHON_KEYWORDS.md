## Python Keywords


| Keyword    | Meaning                     | Example                       |
| ---------- | --------------------------- | ----------------------------- |
| `False`    | Boolean false value         | `flag = False`                |
| `None`     | Represents absence of value | `x = None`                    |
| `True`     | Boolean true value          | `flag = True`                 |
| `and`      | Logical AND                 | `if a and b:`                 |
| `as`       | Alias in import / context   | `import math as m`            |
| `assert`   | Debug check                 | `assert x > 0`                |
| `async`    | Define async function       | `async def fetch(): pass`     |
| `await`    | Wait for async result       | `await fetch()`               |
| `break`    | Exit loop                   | `break`                       |
| `class`    | Define a class              | `class MyClass: pass`         |
| `continue` | Skip iteration              | `continue`                    |
| `def`      | Define function             | `def add(a, b): return a + b` |
| `del`      | Delete object               | `del x`                       |
| `elif`     | Else-if condition           | `elif x == 0:`                |
| `else`     | Default condition           | `else:`                       |
| `except`   | Catch exception             | `except ValueError:`          |
| `finally`  | Always execute block        | `finally: cleanup()`          |
| `for`      | Loop over iterable          | `for i in range(5):`          |
| `from`     | Import specific item        | `from math import sqrt`       |
| `global`   | Declare global var          | `global x`                    |
| `if`       | Conditional                 | `if x > 0:`                   |
| `import`   | Import module               | `import os`                   |
| `in`       | Membership test             | `if x in lst:`                |
| `is`       | Identity comparison         | `if x is None:`               |
| `lambda`   | Anonymous function          | `f = lambda x: x+1`           |
| `match`    | Pattern matching            | `match x:`                    |
| `case`     | Match branch                | `case 1:`                     |
| `nonlocal` | Outer scope var             | `nonlocal x`                  |
| `not`      | Logical NOT                 | `if not x:`                   |
| `or`       | Logical OR                  | `if a or b:`                  |
| `pass`     | No-op placeholder           | `pass`                        |
| `raise`    | Raise exception             | `raise ValueError()`          |
| `return`   | Return from function        | `return x`                    |
| `try`      | Start exception block       | `try:`                        |
| `while`    | Loop with condition         | `while x < 5:`                |
| `with`     | Context manager             | `with open(f) as f:`          |
| `yield`    | Generator output            | `yield x`                     |
